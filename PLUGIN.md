# vein-launch Plugin

## Description

SOTA Claude Code launcher with 7-tier precheck, CLIProxy management, RTK token compression, and GPT-5.5 quality gates.

## Usage

```
vein <project> [--deep | --repair]
```

## Skills

- `/launch` — Run prechecks and launch Claude for a project
- `/status` — Show current environment health
- `/accounts` — Manage CLIProxy accounts
- `/team` — Manage agent team configuration
